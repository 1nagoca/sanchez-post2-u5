# 🛒 Catálogo Web con Carrito de Compras

Este es un proyecto de aplicación web desarrollada en Java que simula un catálogo de productos. Permite a los usuarios visualizar artículos, filtrarlos, agregarlos a un carrito de compras temporal (utilizando sesiones HTTP) y simular el proceso de pago.

## 🚀 Tecnologías Utilizadas
* **Backend:** Java, Jakarta EE (Servlets)
* **Frontend:** JSP, HTML5, CSS3, Expression Language (EL)
* **Servidor Web:** Apache Tomcat
* **Estructura/Build:** Maven

## ⚙️ Requisitos Previos
Para ejecutar este proyecto en un entorno local, asegúrate de tener instalado:
* **Java Development Kit (JDK):** Versión 11 o superior.
* **Apache Tomcat:** Versión 10 o superior (compatible con `jakarta.servlet`).
* **IDE:** IntelliJ IDEA (versión Ultimate recomendada) o similar.
